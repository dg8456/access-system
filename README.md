# Arduino寝室门禁系统
An access system based on Arduino with rifd, bluetooth and servo module

使用到的主要模块：

- Arduino Uno R3
- HC-06蓝牙模块
- MFRC522 射频识别模块 
- MG996R 舵机

详细介绍请参见我的文章：[[Arduino]寝室智能门禁(刷卡+手机蓝牙)](http://imfing.com/2016/08/30/Arduino-smart-lock.html),  [简书](http://www.jianshu.com/p/bbed79b82227)
